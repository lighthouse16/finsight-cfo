Review the finsight-cfo frontend as a senior product designer.

This project is a professional AI CFO / SME finance workspace. It should feel like:
- banking-grade
- enterprise fintech
- calm and trustworthy
- data-driven
- commercially usable

It should not feel like:
- a crypto dashboard
- a colorful startup toy
- a generic admin template
- a chatbot-only demo
- random cards of text

Use the global taste-skill / design-taste principles if available, but adapt them to this product context.

Inspect:
- src/App.tsx
- src/main.tsx
- src/index.css
- src/pages/
- src/components/
- src/data/
- app/
- components/
- styles/

Check:
1. visual hierarchy
2. information architecture
3. card density
4. spacing and alignment
5. typography
6. responsive behavior
7. table/chart readability
8. finance trust language
9. product coherence
10. whether the UI feels like a senior-built product

Return:
1. Overall taste score out of 10
2. Top 5 UI/product issues
3. Exact files/components likely responsible
4. Concrete fixes
5. What to implement first
6. One safe implementation prompt for the frontend engineer
